# Show HN: Praxeos – Explorable explanations for Austrian economics

**Title:** Show HN: Praxeos – Explorable explanations for Austrian economics
**URL:** https://praxeos.org

**Post body (~120 words):**

I've been building an open-source library of interactive, generative
explanations of Austrian economics in the lineage of Bret Victor, Nicky
Case, and Bartosz Ciechanowski. Three modules in Fascicle I:

- **The Halving Garden** — every Bitcoin block from genesis rendered as
  a botanical organism on a Hilbert curve, four halving epochs as
  illuminated-manuscript pages, live mempool.space WebSocket for new
  blocks.
- **The Time Preference Forest** — ABCT as an 18th-century woodcut; a
  correction cascade reveals actual capital depth.
- **The Calculation Problem** — Mises's 1920 argument as a two-panel
  typographic simulation with a "Be the Planner" mode that fails the way
  the argument predicts.

Next.js 15, React 19, Canvas 2D + SVG + Playwright e2e. MIT (code) + CC
BY 4.0 (content). Feedback welcome.
